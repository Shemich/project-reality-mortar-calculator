## How to run
1. Clone
2. Run index.html
3. Choose map
4. Left click - put a target marker, Right click - put a marker of mortar position

## Как заупустить
1. Клонируете
2. Запускаете index.html
3. Выбираете карту
4. ЛКМ - поставить маркер цели, ПКМ - поставить маркер позиции миномета
